The Responsible

An interactive story written by Samuel & Joel Plumppu

This is an open game where your actions matter.

To learn about available actions, type help at any time.

This is a list of available actions, and examples of how to use them.

Actions:

Type keywords as shown in the left column below. Replace any [] with other values.

[direction] Move in a given direction. Each value also has it's first letter as a shorthand. E.g. 
north
is the same as 
n
.
take [object] Pick up an object.
drop [object] Drop an object.
read [object] Read a book or a note.
inspect [object] See how you can interact with an object. Useful if you need a hint for how to continue.

Text highlighting:

Formatted text helps important information to stand out.

Bold Actions, like directions where you can go, or something you can do.
Italic Items or objects that can be interacted with. This is also used to highlight dialogue or your character's thoughts.
keyword Keywords are literal text you can enter as input. These often contain [] which should be replaced by some other value. Luckily, they have a description (E.g. [object]) that tell you what's expected in their place.

Convenience features:

These features let you focus more on the game, and less on typing.

Autocomplete Press TAB to autocomplete the current action or object you're typing. Saves a lot of typos and time!
Autofocus Type anywhere on the page and it'll automatically get into the text input.
Action history Press Up Arrow or Down Arrow to see the previous actions you've used. Simply press enter to reuse a prevoius one!

Please note that these features work best in the game itself.
