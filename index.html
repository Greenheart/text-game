Your story

This is an open game where your actions matter.

To learn about available actions, type help at any time.

Play this game by thinking about the current state of the game and choosing how you want to react. Then type your action into the text input.

Below is a list of available actions, and descriptions how to use them:

[direction]
Move in a given direction.
take [object]
Pick up an object.
drop [object]
Drop an object.
read [object]
Read a book or a note.
