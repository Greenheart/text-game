The Responsible

An interactive story written by Samuel & Joel Plumppu

This is an open game where your actions matter.

To learn about available actions, type help at any time.

Play this game by thinking about the current state of the game and choosing how you want to react. Then type your action into the text input.

Below is a list of available actions, and examples of how to use them.

Commands:

Type keywords as shown in the left column below. Replace contents of [] with other values.

[direction] Move in a given direction. Each value also has it's first letter as a shorthand. E.g. 
north
is the same as 
n
.
take [object] Pick up an object.
drop [object] Drop an object.
read [object] Read a book or a note.
